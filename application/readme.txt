############################
# Description des dossiers #
############################

- configs :
	Contient les fichiers de configuration généraux de l'application (.ini).

- forms :
	Contient les formulaires de l'application.

- language :
	Non utilisé.

- layouts :
	Contient les mise en pages (templates) du site.

- models :
	Contient les classes reletavite à cheque table de la BDD.

- modules :
	Contient le code (PHP et HTML) de l'ensemble des modules du site
	répartis en 3 groupes : "frontend", "backend" et "espclient".

- plugins:
	Non utilisé.

- views :
	Contient des fonctions générales.
